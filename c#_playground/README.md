# OOP Playground: C# Section

## Overview
This folder contains small C# console applications designed to practice **Object-Oriented Programming (OOP)** and core C# concepts.  

The projects here are part of a larger **OOP playground**, which also includes Java exercises. This section focuses on:

- **Data Types**
- **Type Conversion**
- **Loops and Nested Loops**
- **Basic Methods & OOP principles**

These exercises are meant for **beginner to intermediate learners** to build confidence in C# and console programming.

## Loops Project (`Loops.cs`)

Namespace: `CSharpPractice.Loops`

### Features Implemented:
1. **For Loops**
   - Iterates through numbers and prints them to the console.

2. **Multiplication Table**
   - Nested loops generate a multiplication table.
   - Demonstrates control flow and iteration.

### Example Usage:

```csharp
using CSharpPractice.Loops;

Loops.ShowLoops();        // Prints numbers from 0 to 3
Loops.Multiplication();   // Prints a 4x4 multiplication table
````

## How to Run

1. Open terminal/command prompt and navigate to the folder:

```bash
cd OOP/c#_playground/LoopsPatterns
```

2. Run the program using .NET CLI:

```bash
dotnet run
```

> Make sure .NET SDK is installed and environment variables are set.
